# Code of Conduct

This project aims to maintain a professional, respectful and productive environment.
By participating in this repository, you agree to the following:

## Expectations
- Communicate clearly and stay focused on the technical topic.
- Assume good intent but be precise and factual.
- No personal attacks, harassment, trolling, or low-effort engagement.
- Respect maintainersâ€™ time. Poorly prepared issues or PRs may be closed without discussion.

## Maintainer Responsibilities
- Review contributions objectively and transparently.
- Enforce standards consistently.
- Set technical direction and decide what gets merged.

## Scope
Applies to all project spaces: issues, pull requests, discussions, and any community channels.

## Reporting
If there is unacceptable behavior, open a private issue or contact the maintainer directly.

Maintainers reserve the right to remove or ban individuals violating this Code of Conduct.
